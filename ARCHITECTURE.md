# Architecture du Projet Vote CFLB

```
vote-project/
│
├── .gitignore                          # Fichiers ignorés par Git
├── .vscode/
│   └── extensions.json                 # Extensions recommandées pour VSCode
│
├── index.html                          # Point d'entrée HTML de l'application
├── package.json                        # Dépendances et scripts npm
├── package-lock.json                   # Verrouillage des versions des dépendances
├── vite.config.js                      # Configuration Vite
├── README.md                           # Documentation du projet
│
├── public/                             # Fichiers statiques publics
│   ├── _redirects                      # Règles de redirection (Netlify)
│   └── CFLB-logo-rounded.png          # Logo CFLB arrondi
│
└── src/                                # Code source de l'application
    │
    ├── main.js                         # Point d'entrée JavaScript
    ├── App.vue                         # Composant racine de l'application
    ├── style.css                       # Styles globaux
    │
    ├── assets/                         # Ressources statiques
    │   ├── CFLB-logo-bgless.png       # Logo CFLB sans fond
    │   ├── CFLB-logo.png              # Logo CFLB standard
    │   └── main.css                    # Styles principaux
    │
    ├── components/                     # Composants réutilisables
    │   ├── AppLoader.vue              # Composant de chargement
    │   ├── FloatingSearch.vue         # Barre de recherche flottante
    │   └── Navbar.vue                 # Barre de navigation
    │
    ├── services/                       # Services et API
    │   ├── candidateService.js        # Service de gestion des candidats
    │   └── supabase.js                # Configuration Supabase
    │
    ├── router/                         # Configuration du routage
    │   └── index.js                    # Définition des routes Vue Router
    │
    └── views/                          # Vues/Pages de l'application
        ├── AdminDashboard.vue         # Tableau de bord administrateur
        ├── CandidateProfileView.vue   # Profil détaillé d'un candidat
        ├── CandidatesGalleryView.vue  # Galerie de tous les candidats
        └── HomeView.vue               # Page d'accueil
```

## Description des dossiers principaux

### `/public`
Fichiers statiques servis directement par le serveur. Accessibles via l'URL racine.

### `/src`
Code source principal de l'application Vue.js.

### `/src/assets`
Ressources statiques (images, CSS) importées dans les composants.

### `/src/components`
Composants Vue réutilisables dans toute l'application.

### `/src/services`
Services pour interagir avec les APIs externes (Supabase, etc.).

### `/src/router`
Configuration du routage Vue Router pour la navigation.

### `/src/views`
Pages principales de l'application, correspondant aux routes.

## Technologies utilisées

- **Vue 3** : Framework JavaScript
- **Vite** : Build tool et serveur de développement
- **Vue Router** : Routage côté client
- **Supabase** : Backend as a Service (BaaS)

```create table public.candidates (
  id bigint generated by default as identity not null,
  nom text not null,
  photo text not null,
  categorie text not null,
  votes_count integer null default 0,
  created_at timestamp with time zone null default now(),
  constraint candidates_pkey primary key (id)
) TABLESPACE pg_default; 
```

- **Bootstrap 5** : Framework CSS
- **Bootstrap Icons** : Bibliothèque d'icônes
- **KkiaPay** : Intégration de paiement
